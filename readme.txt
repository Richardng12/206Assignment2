Name Sayer
*****************************************************************

This program allows users to create and listen to creations in order to practice pronunciation of names.

*****************************************************************

How to run:

1. Place the Name_Sayer jar file in any specified directory
2. Open up terminal and change your directory to the same place where the jar file is
3. Run the application using the command "java -jar Name_Sayer.jar" (Excluding quotation marks)
